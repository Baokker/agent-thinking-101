# Agent Thinking 101 学习者练习册

这份练习册留给课堂动手和课后复盘。每节课至少写一次自己的输入，也要记录 AI 第一版哪里不对。空着没关系，拿模板照抄就没什么用了。

## 使用方式

每节课做三件事：

1. 写一份自己的输入。
2. 让 AI 给出结果。
3. 写下你怎么检查和修改。

不知道做什么时，就从最近的一件小麻烦开始。个人主页、旅行计划、表格或照片工具都行，前提是你确实想用它。
